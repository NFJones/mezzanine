# Mez Manual

Use this skill when the user asks how to operate Mezzanine itself.

Terminal commands begin with `:` and control the multiplexer session. Agent shell commands begin with `/` and control the active agent pane. Explicit skills are invoked by starting an agent prompt with `$<skill-name>` followed by optional task-specific context.

For operational answers, cite the relevant command names and keep instructions direct. For implementation work, use the manual only as orientation and then inspect the responsible code.
